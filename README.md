# Appiness Tasks

Authentication Front-End and Back-End

## Backend NodeJs
 
Install the Dependencies
```javascript
npm install
```

Run Node Js
```javascript
npm start
```

## Frontend React Native

Install Dependencies
```javascript
npm install
```
Run React Native
```javascript
expo start or npm start
```
